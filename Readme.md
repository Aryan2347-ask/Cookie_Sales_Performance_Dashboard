<h1 align="center">🍪 Cookies Performance Dashboard</h1>

An interactive Excel dashboard project designed to analyze cookie sales performance, profitability trends, and product-wise business insights using advanced Excel features.

---

📌 Project Overview

The **Cookies Performance Dashboard** provides a complete overview of sales and profit performance using interactive charts, slicers, KPIs, and pivot tables.

This dashboard helps businesses:

* Track monthly sales performance
* Monitor profit trends
* Analyze country-wise revenue
* Compare product performance
* Make data-driven business decisions

---

📊 Dashboard Features

✅ KPI Tracking

* Total Revenue
* Total Profit
* Units Sold
* Monthly Performance Analysis

✅ Interactive Elements

* Dynamic Slicers
* Pivot Charts
* Product Filters
* Country Filters
* Monthly Trend Analysis

✅ Business Insights

* Best-performing cookie products
* Profit contribution by country
* Monthly sales growth trends
* Product-wise unit sales comparison
